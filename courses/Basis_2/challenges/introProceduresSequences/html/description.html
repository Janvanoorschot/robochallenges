Sequences of procedures

You can use procedures everywhere. Procedures do not depend on where you should start. That way you can use the same procedure in many different places!

However, if you want to keep track of what Robo will do, it is very handy if he always finishes the procedure with some standard position and orientation.

That way you can easily use procedures in long combinations, like here to write a number down.
