The procedure "turn"

As you saw, you can say "turn" to Robo only if you make a procedure with the name "turn".

However, if you only make a procedure nothing will happen.

You must also give Robo the instruction turn to let him do it.

In the script editor we have already given an example. Push the button to see what happens.

Why don't you make your own program using the procedure turn?
For example to let Robo turn 4 times, or 1 time in each square, or ... .
