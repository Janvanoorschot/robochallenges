Track the white dots

Robo has to find a beacon and pick it up. For this purpose someone has left a trace of white dots which Robo can use to retrieve the beacon. This is quite similar to tracking and tracing! Do you know how Robo can find the beacon?

To see what is in front of the robot you can use, for example, frontIsWhite. This tests if something white is in front and you can use it in combination with if(.){...} else {...} to make the robot do either one thing or the other depending on the conditions he is facing.

Write a program so that the robot will automatically find the beacon by using the white dots. Use repeat{...}, if(.){...} else {...}, and frontIsWhite.

See the documentation for an example to help you on your way.
