Go to the wall and stop

Let the robot go until it reaches the wall and then stop. Do NOT count the number of steps until the wall!

Use repeat{...}, if(.){...} else{...}, and frontIsClear. Use end to stop the robot.

With if(.){...} else{...} a test is done between the brackets (.). For example if frontIsClear. When the test turns out to be positive or true, the commands between the {} brackets following the if(.) statement are executed. Otherwise the robot executes the commands between the {} brackets of the else command.
