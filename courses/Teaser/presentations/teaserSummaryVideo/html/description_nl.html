Dit filmpje laat je de eerste stappen van programmeren zien.

  • Simpele programma's zijn een reeks instructies die kunnen worden uitgevoerd.
  • Je kan een programma maken door de tekst in de script editor te wijzigen.
  • Robo begrijpt niet zomaar alles wat je opschrijft.
  • Om te zien welke instructies kunnen worden gebruikt kan je de afstandsbediening gebruiken of een kijkje nemen onder het kopje "Help".
