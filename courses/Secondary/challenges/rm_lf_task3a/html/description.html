Walking a grid: structured search

Solve the grid with Method 1: structured grid search. Do not use "dead reckoning" to find the black spot. That is: by looking at the map, counting the squares, and then moving east and then south the correct number of squares. This is considered cheating. Write your code so that your robot can find the black spot no matter where it is placed on the grid.

1. Let the robot search the grid line by line to find the black spot.

2. Use procedures to search each line.
