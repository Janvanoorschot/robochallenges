Een grid van witte lijnen

  • De kaart wordt nu wat moeilijker met een grid van witte lijnen.
  • Er bevindt zich ergens op het grid een zwarte stip
  • Jouw taak is om Robo over het grid te laten lopen om de zwarte stip te vinden.

Een grid doorzoeken

  • Voor deze taak moet je een algoritme ontwikkelen.
  • Dat wil zeggen: een algemene oplossing bedenken voordat je begint te programmeren.
Een grid doorzoeken: algoritmes
Twee manieren om deze grid te doorzoeken zijn:
  1. Systematisch het hele grid aflopen. Als je één voor één elke witte lijn afloopt zal je uiteindelijk de zwarte stip vinden.
  2. Willekeurig het grid aflopen. Neem hiervoor op elke kruising een willekeurige beslissing om links, rechts, of vooruit te gaan.
