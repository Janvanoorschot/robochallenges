Lijnenvolgers

Het volgen van lijnen is iets wat veel robots moeten kunnen. Een post-robot volgt
bijvoorbeeld lijnen in een gebouw om van het ene kantoor naar het andere te gaan.

Lijnen volgen

  • Robots die lijnen volgen zie je vaak in de echte wereld
  • Het is daarom ook onderdeel van robot wedstrijden
  • Deze lessen geven je een goede introductie van het soort programmeertaken die je tegen kunt komen als je een robot automatisch lijnen wilt laten volgen

In deze lessen:

  • Krijg je een begrip van algoritmes
    (de set stappen om een probleem op te lossen)
  • Maak je zelf algoritmes om lijnvolgtaken op te lossen
  • Vertaal je die algoritmes in computer code
  • Programmeer je daarmee de robot
