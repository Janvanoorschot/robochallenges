Line followers

Line following is an ability of many robots. For example, mail-delivering robots
follow lines in an office building to get from one office to the next.

Line Following

  • Line following is a real-world robotics application
  • Therefore it is often part of robotics competitions
  • This activity is a good introduction to the programming tasks you might face in order to let your robot follow lines

In this lesson, you will:

  • Develop an understanding of algorithms
    (the steps one takes to solve a problem)
  • Create algorithms to solve line following tasks
  • Turn your algorithms into code for your robot
  • Program the robot
